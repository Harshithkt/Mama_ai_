# MamaAI

MamaAI is an AI-powered maternal healthcare assistant designed for rural Indian mothers and ASHA workers. 

## Structure
- `frontend/`: Vite + React + Tailwind CSS web application.
- `backend/`: Python Flask backend for serving ML APIs.

## Run Frontend
```bash
cd frontend
npm install
npm run dev
```

## Run Backend
```bash
cd backend
pip install -r requirements.txt
python app.py
```
